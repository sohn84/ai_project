# 📋 AI 썸네일 영상 스튜디오 기획 요약서

## 1. 배경 & 문제

### 현재 상황
- 여행 상품 영업팀, 제품 소개 영상 썸네일 필요 (1인당 주 10개, 월 40개)
- 내부 제작 인력 없음 + 외주 비용 부담 (1개당 40만원 = 월 1.6억원)
- 결과: **썸네일을 아예 만들지 못함** → 정적 이미지로 대체

### 핵심 문제
> "전문가 없이도 제품 소개영상 썸네일을 빠르고 저렴하게 만들 수 없다"

### 해결 기회
- AI 영상 생성 기술 성숙 (Google Veo, Luma 등)
- 비용 절감 잠재력: **월 1.59억원 (99.7%)**

> "영업팀 누구나 전문가급 제품 영상을 **5분 안에** 만든다"

2. 월 1억+ 비용 절감
3. 전환율 20%+ 향상 검증

### 핵심 KPI
- 💰 비용 절감액: 월 1.5억원+
- 📈 구매 전환율: 동영상 썸네일 상품 20%+ 향상
- 👥 사용률: WAU 80%+ (10명 중 8명)

- 직급: 대리급, B2C 대고객 영업

- 절실함: ⭐⭐⭐⭐⭐⭐⭐⭐⭐ (9/10)

## 4. 솔루션

### 한 줄 설명
> "여행 상품 영업팀을 위한 AI 기반 8초 썸네일 영상 자동 제작 도구"

### 핵심 차별점
1. **자동 프롬프트 생성**: 이미지만 올리면 AI가 최적 프롬프트 작성
2. **여행 특화**: 여행 상품 이미지 DB 연동
3. **초간단 UX**: 클릭 3번으로 완성

### vs 범용 도구 (Runway/Pika)
| 항목 | 범용 도구 | 우리 도구 |
|------|-----------|-----------|
| 프롬프트 | 직접 작성 필요 | **자동 생성** |
| 품질 일관성 | 낮음 (사용자 역량) | **높음 (AI 최적화)** |
| 여행 맥락 | 없음 | **있음 (DB 학습)** |
| 제작 시간 | 30분+ | **5분** |

## 5. 핵심 기능

### 사용자 여정 (7단계)

```
1. 홈 화면
   ├─ 이전 작업물 리스트
   └─ [+ 새 영상 만들기]

2. 이미지 업로드
   └─ 📸 사진 4장 선택

3. AI 분석
   └─ 🤖 이미지 분석 중...

4. 프롬프트 생성 ← ✨ Aha Moment!
   ├─ 자동 생성된 프롬프트 표시
   └─ 수정 가능 (선택)

5. 모델 선택
   ├─ Google Veo
   └─ Luma Dream Machine

6. 영상 생성
   └─ 🎬 생성 중... (2-3분)

7. 완료
   ├─ ▶️ 재생
   └─ [다운로드]
```

2. ✅ AI 이미지 분석 (GPT-4 Vision)
3. ✅ 프롬프트 자동 생성
4. ✅ 모델 선택 (Veo/Luma)
5. ✅ 8초 영상 생성
6. ✅ 재생 & 다운로드

### 향후 추가
- 📝 텍스트 오버레이
- 🎵 배경음악
- ✂️ 길이 조절
- 🎨 브랜드 로고
