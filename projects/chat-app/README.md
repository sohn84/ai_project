# Chat App

OpenAI ChatKit을 활용한 채팅 애플리케이션입니다.

## 기술 스택

- **Framework**: Next.js 15
- **Language**: JavaScript/React
- **UI Library**: @openai/chatkit-react
- **Runtime**: React 19

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3001](http://localhost:3001)을 열어 확인하세요.

> 이 프로젝트는 포트 3001에서 실행됩니다.

### 빌드

```bash
npm run build
npm start
```

### 린트

```bash
npm run lint
```

## 주요 기능

- OpenAI ChatKit을 활용한 채팅 인터페이스
- React 19 기반의 모던 UI
