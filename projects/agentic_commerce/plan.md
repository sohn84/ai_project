현재 단계를 진행하면 이전 단계의 상품이나 CTA는 모두 삭제되는 형태로 UX를 구성한 이유
1. 챗봇 대화 흐름의 자연스러움 - 일반적인 챗봇 UX에서는 대화가 진행되면서 새로운 컨텍스트로 넘어갈 때 이전 인터랙티브 요소들이 비활성화되거나 사라지는 것이 자연스럽습니다.
2. 사용자 집중도 유지 - 각 단계마다 사용자가 취해야 할 액션에 집중할 수 있도록 현재 단계의 콘텐츠만 활성화 상태로 유지합니다.
3. 화면 복잡도 관리 - 모든 단계의 상품 카드와 CTA 버튼이 누적되면 스크롤이 매우 길어지고, 어떤 버튼을 클릭해야 하는지 혼란스러워질 수 있습니다.
4. 모바일 최적화 - 390px의 좁은 모바일 화면에서 누적된 콘텐츠로 인한 스크롤 피로도를 줄입니다.
5. 명확한 단계 진행 - 예약 플로우가 단계별로 명확하게 진행되고 있다는 것을 시각적으로 표현합니다.
다만, 이전 단계로 돌아가거나 다른 상품을 다시 보고 싶을 때의 경험은 제한적일 수 있습니다.
